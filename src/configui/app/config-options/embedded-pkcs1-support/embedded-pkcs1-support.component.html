Starting from Node.js versions 18.19.1, 20.11.1, and 21.6.2, the removal of RSA_PKCS1_PADDING support breaks Eufy Security's livestream/P2P functionality. We've developed a countermeasure based on software calculation, it will be 10x times less effective than hardware but it can make some cameras work on Node.js 22+. More technical details here.
