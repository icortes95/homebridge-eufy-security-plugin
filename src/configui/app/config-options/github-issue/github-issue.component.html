To troubleshoot effectively before rising your issue to us:

  1. Search Existing Issues: Check Common Issues and User-Raised Issues.
  2. Join Discord Server: Engage with developers and community on Discord.
  3. Enable debug logging
  4. Reboot your system
  5. Reproduce the issue
  6. Download logs

Resetting the system may also help.

Remember to disable debug logging after troubleshooting.
